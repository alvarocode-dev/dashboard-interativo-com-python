📌 Sobre o Projeto
Este projeto é um Dashboard Interativo desenvolvido em Python com o objetivo de transformar dados em informações visuais claras e intuitivas.
A aplicação permite explorar dados de forma dinâmica por meio de gráficos interativos, filtros e métricas atualizadas automaticamente.
O projeto foi criado para praticar análise de dados, visualização e desenvolvimento de aplicações em Python.

🚀 Tecnologias Utilizadas

Python 3
Pandas
Plotly
Streamlit
NumPy

📈 Funcionalidades

Visualização interativa de gráficos
Filtros dinâmicos
Indicadores e métricas automáticas
Interface simples e intuitiva
Análise exploratória de dados
